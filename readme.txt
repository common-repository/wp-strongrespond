== Description ==

Ce plugin va mettre en gras les pseudos des visiteurs auxquels vous apportez une réponse dans les commentaires. 

Dès que vous tapez "@pseudo" dans la case des commentaires pour faire une réponse, le mot qui sera derrière le "@" sera en gras, ce qui apporte une meilleure visibilité pour la personne concernée.

== Installation ==

1. Upload `WP-StrongRespond.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy ;)
